# ReactNative 仿豆瓣-Android (2018年8月安卓版)
## 运行
 - 先确保你已安装好了React Native 所需的依赖环境 并且连接移动(虚拟)设备
 - 在根目录下执行 npm install
 - 在执行 react-native run-android
